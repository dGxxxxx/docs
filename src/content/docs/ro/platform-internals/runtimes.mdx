title: Runtimes
---

YAMP suportă mai multe runtime-uri de scripting, permițând dezvoltatorilor să aleagă limbajul care se potrivește cel mai bine preferințelor și nevoilor proiectului lor.

## Runtime-uri Oficial Suportate
*   **Lua:** Un limbaj de scripting ușor, integrabil, cunoscut pentru simplitatea și viteza sa. Adesea preferat pentru dezvoltare rapidă și logica comună a jocului.
*   **JavaScript (JS):** Un limbaj versatil, folosit pe scară largă, cu un ecosistem vast. Potrivit pentru diverse sarcini, inclusiv dezvoltarea UI atunci când este folosit împreună cu framework-uri compatibile.

Aceste runtime-uri oferă API-uri pentru a interacționa cu funcționalitățile platformei YAMP (entități, evenimente, natives etc.).

## Extensibilitate prin C SDK
Dincolo de runtime-urile oficiale, YAMP pune la dispoziție un **C SDK** de bază. Acest SDK expune API-urile platformei, permițând comunității să:  
*   Dezvolte și integreze suport pentru limbaje suplimentare (de ex., Python, Rust, Go).  
*   Creeze runtime-uri personalizate cu funcționalități sau integrări unice.

Aceasta încurajează un ecosistem extensibil în care comunitatea poate contribui semnificativ la capabilitățile platformei. (Vezi [SDK Development](../sdk-development/introduction.md))
