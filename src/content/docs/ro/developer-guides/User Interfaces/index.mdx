YAMP oferă mai multe soluții pentru construirea interfețelor utilizator, adaptate nevoilor diferite privind performanța și funcționalitățile.

*   **Ultralight:** Destinat ca opțiunea principală de înaltă performanță, potrivit pentru elemente UI solicitante unde eficiența este critică. (Vezi [Ultralight Guide](./ultralight-guide.md))
*   **CEF (Chromium Embedded Framework):** O alternativă mai bogată în funcționalități și stabilă, bazată pe Chromium, potrivită pentru interfețe complexe care pot semăna cu paginile web. Are în general un overhead de performanță mai mare decât Ultralight. (Vezi [CEF Guide](./cef-guide.md))

Scopul este să oferim dezvoltatorilor toolkit-uri UI axate pe performanță și ușurință în dezvoltare, care să se integreze perfect cu runtime-urile disponibile. Considerațiile cheie la alegere includ impactul asupra performanței, ușurința în dezvoltare, setul de funcționalități (suport HTML/CSS/JS) și compatibilitatea cross-platform.

Consultați ghidurile specifice pentru detalii despre API-ul și utilizarea fiecărei soluții. ([UI Best Practices](./ui-best-practices.md))
